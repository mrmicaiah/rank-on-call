# rank-on-call

Bound to Studio87 on 2026-07-21.

## Project intent

Rank On Call is a national, self-serve web product that sells a paid "Deep Dive" report — a business-intelligence audit of a contractor's online presence — for $39 (intro price, against a $249 stated value). It has no geographic tether: it runs on public data and never meets the customer, so it sells to contractors anywhere in the US. It is both a profit center and the warm-lead engine for the sister business Sites On Call (websites + SEO), replacing the dead cold-call channel. The funnel is a **content engine** built on pain-symptom keywords ("why isn't my phone ringing") — **not** city pages, which are the Sites On Call model and fail here.

See `docs/PROJECT_MASTER.md` for the full master document.
